# Research Wiki: EEG Seizure Detection

This wiki accumulates knowledge from papers, code, and experiments on LSTM-based seizure detection from EEG signals.

**Start here:** See `QUICKSTART.md` in the project root.

## Contents

- **index.md** — Catalog of all pages (updated on each ingest)
- **log.md** — Chronological record of additions and updates
- **entities/** — Core concepts (models, datasets, metrics, techniques)
- **topics/** — Research areas
- **sources/** — Summaries of ingested papers
- **syntheses/** — Cross-cutting analyses, comparisons, benchmarks

## Workflow

1. **Find a paper** (arxiv-sanity-lite, arXiv, etc.)
2. **Add to `../sources/<topic>/`** with notes
3. **Say "Ingest `<path>`"** 
4. **Review the generated wiki pages**
5. **Commit changes**

The wiki grows incrementally. Each paper updates related entity pages, strengthens connections, surfaces contradictions.
