# Reports — spring-2026-group2

## Structure

```
reports/
├── Latex_report/       ← LaTeX formatted final report
├── Markdown_Report/    ← Markdown formatted reports and summaries
├── Word_Report/        ← Word/PDF formatted reports
└── Progress_Report/    ← Weekly progress reports
```

## Submission Guidelines
- Progress reports submitted weekly in `Progress_Report/`
- Final report available in LaTeX, Markdown, and Word formats
- All figures referenced in reports go into the respective `fig/` subdirectory
